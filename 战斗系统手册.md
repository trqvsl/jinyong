## 二、回合流程

```
玩家回合（playerTurn）
  └─ 玩家选招 → handlePlayerSkill
       └─ performPlayerSkill(纯函数) → 算伤害/状态
       └─ playSkillFlash + addFloat   → 演出动画
       └─ checkBattleEnd → 未结束则 startEnemyTurn

敌人回合（busy → startEnemyTurn）
  ├─ tickStatuses(敌人) / tickStatuses(玩家)   状态结算
  │    └─ checkBattleEnd（状态可能致死）
  ├─ isStunned(敌人)? → 眩晕则跳过，交还玩家回合
  ├─ enemyChooseSkill → AI 选招
  ├─ performEnemySkill(纯函数) → 算伤害/状态
  │    └─ 演出 → checkBattleEnd → 回 playerTurn
```

**关键认知**：状态结算（中毒掉血/增益递减）发生在**敌人回合开头**（`startEnemyTurn`），玩家回合开始时身上状态还没 tick。这是当前时序特征，改机制时要留意。

## 三、根基属性体系（定稿）

> 武侠 RPG 的灵魂：玩家不直接堆"攻击力/血量"，而是修炼**根基属性**；战斗属性由根基**经公式推导**而来。你练的是根基，战斗力是根基长出的枝叶。

### ① 八大根基属性

| 根基 | 武侠意涵 | 主导作用 |
|---|---|---|
| **力量**（Strength） | 筋骨之力 | 外力根基，参与共构攻击力 |
| **外功**（External） | 外家修为 | 与力量共构物理攻击力 |
| **内功**（Internal） | 内家真气 | 经"内功催动"叠加到攻击力 |
| **悟性**（Comprehension） | 灵慧天赋 | 武功修炼速度（熟练度成长）+ 高阶武学门槛 |
| **身体**（Constitution） | 体魄根骨 | 气血上限、防御 |
| **吐纳**（Breath） | 内息运转 | 内力上限 |
| **身法**（Agility） | 敏捷闪转 | 速度、命中、暴击、闪避（四用，控数值防膨胀） |
| **福缘**（Luck） | 气运天命 | 奇遇触发、剧情检定、闪避/命中修正、逃跑成功率 |

> 原 `aptitude`（资质）重命名为**悟性**，不引入冗余属性。

### ② 推导公式树（根基 → 战斗属性）

```
[攻击力] 外功物理攻击 = 力量 × 1.2 + 外功 × 1.5 + 等级×2
          内功催动加成 = 内功 × 1.0 × 招式.innerScale   (见⑤)
          最终攻击 = 外功物理攻击 + 内功催动加成

身体 × 20 + 等级×15 + 50    → 气血上限 hpMax
身体 × 0.8 + 等级×1         → 防御 def
吐纳 × 8 + 等级×5 + 20      → 内力上限 mpMax
身法 × 1.0 + 等级×0.5       → 速度 spd
身法 × 0.6 + 等级×1         → 命中 hit
身法 × 0.2%                 → 暴击率 crit（身法 50 → 10%）
身法 × 0.15%                → 闪避 dodge（身法 50 → 7.5%）
悟性                        → 熟练度成长倍率 = 1 + 悟性/200；高阶武学门槛
福缘                        → 闪避/命中修正 = 福缘×0.1%；逃跑率 = 基础 + 福缘×0.3%；奇遇/剧情检定
```

### ③ 攻击力的三重共构（核心）

```
外功物理攻击 = 力量 × 1.2 + 外功根基 × 1.5 + 等级×2
内功催动加成 = 内功根基 × 1.0 × innerScale
最终攻击力   = 外功物理攻击 + 内功催动加成
```

- **力量**：纯物理蛮力，所有外功的底子，不可替代
- **外功**：外家招式修为，体现"练了多少外家功夫"
- **内功**：内家真气，通过催动系数选择性加持——内功深的人打依赖内力的招式更强

> 让"纯外家高手"（力量+外功高、内功低）和"内外兼修"（三项均衡）都是可行路线。

### ④ 属性层级总图

```
[根基属性]  力量/外功/内功/悟性/身体/吐纳/身法/福缘   ← 玩家修炼的主对象
     │ 推导公式（②）
[战斗属性·面板] 攻/防/血/蓝/速/命/暴/闪            ← 角色页显示
     │ + 装备加成
[战斗属性·最终]                                      ← 公式实际使用
     │ + 战斗中状态（增益/削弱/护盾）
[最终属性]                                            ← 进入总伤害公式（第四节）
```

### ⑤ 内功催动外功（innerScale）

内功根基通过招式的 `innerScale`（对内力依赖度，0~1，默认 0.3）叠加到外功攻击。

| 招式 | innerScale | 说明 |
|---|---|---|
| 太祖长拳 | 0.3 | 纯外家，不太吃内力 |
| 独孤九剑 | 0.5 | 重招式不重力道 |
| 降龙十八掌 | 1.0 | 至刚至阳，极吃内力 |
| 六脉神剑 | 1.0 | 以内力化剑气 |

**数据**：Skill 加 `innerScale: number` 字段（0~1）。

### ⑥ 属性点获取与分配

- **升级**：每升一级获得 3~5 点属性点，玩家手动分配到八大根基
- **秘籍修炼**：练成秘籍/拜师学艺可能直接提升特定根基（如《易筋经》+身体+吐纳）
- 手动分配是武侠 RPG 配点乐趣的核心

### ⑦ 福缘的多功能化

| 用途 | 机制 |
|---|---|
| 奇遇触发 | 提升稀有奇遇/秘籍事件权重 |
| 闪避/命中修正 | 福缘 ×0.1%，叠加在身法之上 |
| 逃跑成功率 | 基础率 + 福缘 ×0.3% |
| 剧情检定 | 关键抉择/隐藏掉落掷骰 |

## 四、战斗公式系统（定稿）

### ① 总伤害公式

```
最终伤害 = [ 基础伤害 ]
          × [ 克制因子 ]      属性相克（见③），0.75~1.5
          × [ 熟练度因子 ]    招式成长，1.0~1.3（满熟练 +30%）
          × [ 暴击因子 ]      暴击 1.5x，否则 1.0
          × [ 随机浮动 ]      ±15%
          - [ 护盾吸收 ]      先吃护盾
```

**基础伤害**：`技能威力 + 攻击方.最终攻击 - 防御方.最终防御`
（外功威力=伤害加成；奇门=毒/控基础；内功=治疗基数）

**为什么乘法链**：让每个因子都能制造显著差异（1.2×1.3=1.56），克制/熟练度/暴击的收益独立可感，玩家能复盘"这刀为啥疼"。

### ② 行动顺序轴（CTB · 规划中）

取消严格轮流。各单位累积**行动值 AT**，先满 100 者行动，行动后 AT-=100（保留余量，快者偶有连动）。

```
每刻：单位.AT += 单位.最终速度
某单位 AT ≥ 100 → 行动，AT -= 100
```

界面顶部画时间轴，玩家能看见未来出手顺序。速度从"只影响闪避"变成"决定出手次数"的一等属性；眩晕=清零并后延。多单位天然支持群攻/队友。

**落地**：重构 `startEnemyTurn` 时序；状态结算改为"某单位行动前 tick 它自己的状态"。

### ③ 属性克制图

利用 `DamageType`（拳掌/刀法/剑法/指法/暗器/内功），行攻列防=克制因子：

| 攻＼防 | 拳掌 | 刀法 | 剑法 | 指法 | 暗器 | 内功 |
|---|---|---|---|---|---|---|
| **拳掌** | 1.0 | 0.9 | 0.75 | 1.1 | 1.0 | 1.1 |
| **刀法** | 1.1 | 1.0 | 0.9 | 1.0 | 1.25 | 0.9 |
| **剑法** | 1.25 | 1.1 | 1.0 | 0.9 | 1.0 | 0.9 |
| **指法** | 0.9 | 1.0 | 1.1 | 1.0 | 1.25 | 1.0 |
| **暗器** | 1.0 | 0.75 | 1.0 | 0.9 | 1.0 | 1.1 |
| **内功** | 0.9 | 0.9 | 0.9 | 0.9 | 0.9 | 1.0 |

**口诀**：剑克拳（兵器克空手）、刀克暗、指克暗、拳克指、暗克内；内功行/列近 1.0（内功走回血/护盾，不参与克制）。

### ④ 招式成长（熟练度）

武功熟练度 0~100。每用一次积累，熟练度越高伤害加成越大，达阈值可领悟变招。

```
熟练度因子 = 1 + 熟练度/100 × 0.3   （满级 +30%）
```

成长按"使用次数 + 对手强度"累积，碾压低级怪几乎不涨。领悟：熟练 100 的降龙十八掌→领悟「亢龙有悔」。

**数据**：Skill 加 `mastery`；Player 记录 `{[skillId]: 次数}`。

### ⑤ 装备系统（规划中）

装备槽：武器/防具/饰品。三类影响：①数值加成（进装备层）②招式专属加成（打狗棒→打狗棒法+15%）③特殊特效（软猬甲反弹 20%）。获取：商店/剧情掉落/奇遇。

### ⑥ 内功/外功融合（后期复杂机制 · 占位）

未来不应"内功外功练得越多越好"，而应有**能否融合 + 融合程度**考量。某些组合可融合出新效果（九阳+降龙→灼伤），需满足条件（招式熟练度、内外平衡），融合程度影响加成上限。基础战斗稳定后再做。

## 五、状态系统（七种状态）

类型定义 `types/index.ts` 的 `StatusKind`；结算 `battle/engine.ts` `tickStatuses`。

| 状态 | tick 时效果 | 来源 |
|---|---|---|
| `poison` 中毒 | 每回合掉 potency 血 | 奇门(千蛛万毒手/灵蛇拳) |
| `heal` 回春 | 每回合回 potency 血 | 内功(九阳神功) |
| `buff-atk` 攻击强化 | 不产数值，只递减回合 | 内功(乾坤大挪移) |
| `buff-def` 防御强化 | 同上 | 内功(易筋经) |
| `buff-spd` 身法提升 | 同上 | 轻功(凌波微步/梯云纵) |
| `stun` 眩晕 | 跳过该单位下回合 | 奇门(摄魂/兰花拂穴) |
| `shield` 护盾 | dealDamage 时抵消伤害 | 内功(北冥神功) |

**已知隐患**：`applyStatus`（`battle/engine.ts`）的覆盖判断对 potency=0（眩晕）判定异常，需改显式判断。

## 六、武功分类与分流

位置 `battle/adapter.ts` `performPlayerSkill` / `battle/adapter.ts` `performEnemySkill`（镜像）。

| 类别 | 行为 | 典型武功 |
|---|---|---|
| 外功 | 走伤害公式直接打伤害，可附带 effect | 降龙十八掌、独孤九剑、六脉神剑 |
| 奇门 | 少量伤害+施加状态 | 千蛛万毒手、化骨绵掌、摄魂/兰花 |
| 内功 | 给自己挂状态（回春/增益/护盾） | 九阳神功、易筋经、乾坤大挪移、北冥神功 |
| 轻功 | 给自己挂身法增益 | 凌波微步、梯云纵 |

分流按 `skill.category`，外功/奇门对敌人，内功/轻功对自己。
